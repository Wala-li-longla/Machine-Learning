hexo+github+zeabur 搭建个人博客网站

Hexo官网

  • Hexo 是一个博客框架。
  • Hexo 使用 Markdown（或其他标记语言）解析文章，生成静态网页，并自带靓丽的主题。

Zeabur官网

博客网站教程

一. Hexo用法

1. 打开命令提示符（win+R-输入cmd）

2. 使用 npm / cnpm 安装 Hexo

3. 使用 hexo 命令 - hexo init 初始化目录

（1）创建文件夹 hexo

（2）进入 hexo 文件夹，地址栏输入 cmd，然后输入 hexo init

（3）或者可以在一些编程软件中（如 VS code）新建终端，然后输入 hexo init

执行一段时间后，会卡住不动，这是因为初始化目录的安装包需要连接外网，所以直接 Ctrl+c 退出，再继续执行 cnpm install 即可

4. 输入命令 hexo g

生成静态文件，会在当前目录下生成一个新的叫做public的文件夹

5. 输入命令 hexo s

启动本地web服务，用于博客的本地预览

  • 执行该命令后，会生成一个地址，复制到浏览器打开，会看到hexo文件夹初始化后的界面

  • 这只是一个基础框架，还未使用任何主题

6. 在Hexo官网主题一栏，搜索Butterfly

（1）点击该主题会进入到github源码

（2）往下滑到以下界面，复制克隆地址，在命令行输入地址，下载该主题（记得关掉网络加速）

（3）再往下滑，执行以下命令，对主题进行配置（使用cnpm）

（4）该主题已经下载到了Hexo的themes文件夹下，如果想要使用该主题：

  1. 打开Hexo下的_config.yml文件

  2. 把landscape改成butterfly

  3. 执行 hexo cl 清除 hexo g 生成的本地缓存，重新执行 hexo g、hexo s，打开网页查看

  4. 要进一步完善该主题，可以点击下面的参考文档链接

  5. 在hexo文件夹创建一个_config.butterfly.yml，然后把themes文件夹下Butterfly文件夹里的_config.yaml里面所有内容复制到_config.butterfly.yml里

  6. 更改 _config.butterfly.yml 的背景图片：index_img

    • 网络链接
      网上找的一个图片
    • 本地图片相对路径
      图片放在themes/butterfly/source/img，但相对路径从img算起
      发布到github上后，这个img文件夹里的图片会上传到github库里的img文件夹，对应着public/img

    执行 hexo cl、hexo g、hexo s

  7. 更改 hexo 文件下的 _config.xml 的 title 和 author

二. 发布到Github

1. 创建github仓库Hexo-Blog

2. config.yaml配置github仓库

3. 上传到github仓库

（1）先执行npm install --save hexo-deployer-git
（2）再依次执行hexo cl、hexo g、hexo d
（3）hexo d 可能会失败，可以执行git config –global http.sslVerify false，再执行hexo d
（4）结束后记得执行git config –global http.sslVerify true

hexo目录和github上的目录是不一样的。
public文件夹对应github库；
发布到github上之后，source文件夹下的内容会更新到public里面。

三. 使用 Zeabur 部署到互联网

1. 进入zeabur官网，得到github授权

2. 新建项目，配置github仓库hexo-blog

3. 点击网络，自动生成一个免费的域名（一个仓库对应一个域名）

4. 访问该域名，即可访问本地hexo文件夹里的内容

因为 zeabur 得到了 github 的授权，所以只要执行 hexo d 更新 github 库，访问 zeabur 生成的域名，显示的内容就会更新

四. Obsidian+Hexo

  1. 创建Obsidian新库“Hexo Blog”
  2. 在Hexo Blog中创建文件夹“机器学习”
  3. Hexo初始化“机器学习”文件夹
  4. 配置butterfly主题
  5. 创建github仓库Machine-Learning
  6. 配置github仓库
  7. 上传到github仓库
  8. zeabur创建项目，配置github仓库
  9. 生成域名oldb.zeabur.app
  10. 在source/_posts下创建笔记”搭建个人博客网站“

五. 本地图片如何处理

图片插入方式

注意：图片路径不能以"\"或者"/"开头

全局资源文件夹（主要方式）

资源（Asset）代表 source 文件夹中除了文章以外的所有文件，例如图片、CSS、JS 文件等。
在 source 文件夹下创建 images 文件夹中，然后通过![](images/image.jpg) 的方法访问。
还可以在 images 文件夹下再创建一个和文章同名的文件夹，把图片分类存放。

文章资源文件夹

如果想要把图片等资源按照文章分类，可以执行以下步骤：

  1. 将 config.yml 文件中的 post_asset_folder 选项设为 true 
  2. 运行cnpm install https://github.com/CodeFalling/hexo-asset-image --save下载插件
  3. 运行hexo n "xxx"，source/_posts文件夹内会生成 xxx.md 文件和一个同名的文件夹
  4. 将图片存放在xxx文件夹下，然后在xxx.md文件里通过![](image.jpg)引用该图片
